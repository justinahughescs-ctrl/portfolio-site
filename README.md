# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my software engineering projects, technical skills, and experience.

## Overview

This project is a clean and scalable front-end portfolio designed using HTML, CSS, and JavaScript. It serves as a central hub for presenting my work and provides a foundation for future full-stack enhancements.

The site focuses on:
- Clear project presentation
- Modern UI/UX design
- Maintainable and extensible structure

---

## Tech Stack

- **HTML5** – Structure and content
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Interactivity

---

## Features

- Responsive multi-page layout
- Clean and modern design
- Reusable components (navbar, cards, sections)
- Project showcase section
- About and contact pages
- Smooth scrolling navigation

---

## Project Structure
portfolio-site/
│── index.html # Entry point and main landing page
│── about.html # Overview of background, skills, and experience
│── projects.html # Displays portfolio projects and descriptions
│── contact.html # Contact details and external links
│── style.css # Centralized styling for all pages
│── script.js # Handles basic interactivity and navigation behavior
